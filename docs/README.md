# Docs Guide

Use this folder for the small set of documents needed to run, test, and debug the local pipeline.

## Read Order

- Start with [README.md](../README.md) for entrypoints and the repo map.
- Then use the matching file under [`operational/`](./operational/).

## What Lives Here

- `operational/ORCHESTRATION.md`: canonical stage order and rerun scope
- `operational/DATA_CONTRACT.md`: file and schema contracts
- `operational/CODEBOOK.md`: labeling definitions
- `operational/RUNBOOK.md`: execution aid
- `operational/TASKS.md`: current implementation checklist

## Ignore When Navigating

- generated reports under `data/analysis/`
- `__pycache__/`
